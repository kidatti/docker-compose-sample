docker、docker-compose のインストールが必要です

- MySQL 5.6

## [mysql5.7](./mysql5.7)

## [mysql5.7-w-sync](./mysql5.7-w-sync)

- DB

  - データベースファイルの永続化
  - 初期SQL実行
- FILE
  - docker-sync によるホストコンテナ間の高速化
